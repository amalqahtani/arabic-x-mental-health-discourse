# Dataset

The full tweet text cannot be redistributed under the X (Twitter) Developer Policy. This repository provides `post_ids.csv`, containing the tweet IDs and community labels for all 9,582 preprocessed posts described in the paper.

## File Format

| Column      | Description                          |
|-------------|--------------------------------------|
| `post_id`   | Unique X post identifier             |
| `community` | Source community: BPD, bipolar, ADHD |

## Statistics

| Community | Posts |
|-----------|-------|
| BPD       | 6,324 |
| bipolar   | 2,668 |
| ADHD      |   590 |
| **Total** | **9,582** |
